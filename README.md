# Online Process Automation System (OPAS)

OPAS is a centralized, role-based academic workflow automation system designed to eliminate manual paper-based approval processes within a college environment.

## Technology Stack
- **Frontend**: HTML5, CSS3, JavaScript (ES6), Bootstrap 5
- **Backend**: Java (Spring Boot Framework)
- **Database**: MySQL
- **Architecture**: MVC

## Project Structure
- `client/`: Frontend application
- `server/`: Backend Spring Boot application
- `docs/`: Documentation and design resources
